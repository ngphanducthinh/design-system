## Quick Start

**1. Import the styles** in your app entry point:

```ts
import '@7pmlabs/design-system/style.css';
```

**2. Use components** anywhere in your app:

```vue
<script setup lang="ts">
import { BButton, BInput, BModal } from '@7pmlabs/design-system';
</script>

<template>
  <BButton variant="primary">Click me</BButton>
</template>
```
