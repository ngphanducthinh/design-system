# [7PMLabs Design System](https://www.npmjs.com/package/@7pmlabs/design-system)

UI Library with beautifully handcrafted Vue Components for modern web applications. Built for flexibility, accessibility, and rapid development.

---

## ✨ Features
- 🎨 Beautiful, customizable Vue components
- ⚡ Tailwind CSS compatible
- 🔌 Easy integration and usage
- ♿ Focused on accessibility and performance

---

## 🧑‍💻 Creator
Inspired by leading design systems and developed by **Thinh Nguyen**

---

## 🚀 Getting started

```
npm i @7pmlabs/design-system
```

## 🛠️ Usage

Import CSS & plugin in main.ts:

```
import '@7pmlabs/design-system/style.css';
import ds from '@7pmlabs/design-system';

const app = createApp(App);
...
app.use(ds, { i18n });
```

Import & use component:

```
import { BButton } from '@7pmlabs/design-system';

<BButton>Click me!</BButton>
```

## 🎨 Color Customization

CSS Variables:

```
--ds-colors-primary-f: 6 182 212; // #06b6d4
--ds-colors-primary-t: 59 130 246; // #3b82f6
--ds-colors-focus: 86 221 214; // #56ddd6

--ds-colors-primary-f-stop: 0%;
--ds-colors-primary-t-stop: 100%;
```
